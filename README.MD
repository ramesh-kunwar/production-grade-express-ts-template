# Steps To Start the Starter Project

1. Clone the project

```bash
 git clone https://github.com/ramesh-kunwar/production-grade-express-ts-template.git <Project_Name>
```

2. Move into the folder

```bash
cd <Project_Name>
```

3. Install Dependencies

```bash
npm install
```

4. Create new .env file in root directory

5. Start the express server

```bash
npm run dev
```
